# tetris-game-w-github-copilot

Simple Python Tetris using pygame.

Requirements
 - Python 3.8+
 - pip

Quick start (Windows PowerShell):

```powershell
# create venv (optional)
python -m venv .venv; .\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python tetris.py
```

Run unit tests (rotation logic):

```powershell
python -m pip install -r requirements.txt
pytest -q
```

Controls:
 - Left / Right arrows: move
 - Up arrow: rotate
 - Down arrow: soft drop
 - Space: hard drop
